Como surgiram os computadores e sua evolucão

Com a criação dos primeiros computadores: Harvard Mark I, Colossus e ENIAC, feitos a partir de componentes eletromecânicos, foi possível realizar cálculos realmente complexos em um intervalo pequeno de tempo, uma coisa jamais vista na época. A partir dai, os computadores foram evoluindo e seu tamanho cada vez mais reduzido, com a invenção dos componentes eletrônicos: Transistores, CIs (circuitos integrados), PCs (personal computer).

Há quem diga que estamos na 5ª Geração, que envolve algumas tecnologias atuais, como as criptomoedas, IAs e computadores quânticos. Além dos computadores, foram descobertas maneiras de comunicações entre eles, com início na DARPA, depois redes especializadas que usavam protocolos de comunicação mais elaborados. Para chegar onde estamos hoje, alguns nomes tiveram grande participação nessa empreitada e não podemos esquecer de citar:

  • Charles Babbage
  • Joseph Licklider
  • Ada Lovelace
  • Grace Hopper
  • Alan Turing
  • Robert Kahn
  • Tim Berners-Lee

Client na Web

Client são os dispositivos que consomem a informação que está sendo trasmitido via internet, por exemplo, usuarios de navegadores. Essa informação toda vez que é requisitada pelo client, é pedido para o Server mandar essas informações, e através da internet como meio de comunicação, as informações são enviadas ao client.

Os navegadores são aplicativos para abrir e executar os arquivos, seguem padrões W3C e podem ter diferentes interpretações.Resumindo, eles são ferramentas para se utilizar a internet.

Aplicações WEB usam a internet para buscar informações, porém elas não necessariamente devem ser instaladas no dispositivo para conseguir utiliza-las.

Servers

Servers são computadores mais robustos que se comunicam com os outros computadores. Eles provém informações para o client ou outros servers.Existem alguns tipos deles: Arquivos;Segurança;Streaming;E-mail;WEB;

Cada um tem uma caracteristica que os diferenciam dos demais. No caso do WEB, eles hospedam sites.

Linguagens de Programação

Client-side

As mais famosas linguagens voltadas ao Client são as seguintes: HTML (Hyper Text Markup Language); CSS (Cascading Style Sheet) e JavaScript;

Server-side

As mais famosas linguagens voltadas ao Server são as seguintes: PHP, Java, Python e JavaScript;
Obs: JavaScript serve para os dois tipos.

HTML e suas tags

HTML não é uma linguagem de programação, e sim uma linguagem de marcação de texto. Como fazemos para utiliza-la? Simples. Você utiliza as tags para moldar o corpo e título da página, assim podemos realizar diversos tipos de coisas, desde inserir imagens até inserir links, que é uma das principais funções. A maioria das tags voce abre e fecha, e outras como input que somente permite abertura.
